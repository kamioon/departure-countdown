ESP32-S3 · NS API · FreeRTOS

Never miss your train again.

A real-time departure countdown that tells you exactly when to leave home — live from the Dutch Railways API, shown on a dot-matrix display.

GitHub ▶ Watch Demo

Features

  • Live MM:SS countdown — updated every second, DST-aware Amsterdam timezone
  • Walk & Bike modes — alternates every 5 s, switchable via button or web UI
  • Urgent animation — animated icon + flashing timer when <60 s remain
  • Non-blocking NS API fetch — background FreeRTOS task polls every 2 minutes
  • Built-in web UI — configure from any browser on your local network
  • REST API — CORS-enabled JSON endpoints for Home Assistant / Node-RED
  • Secure credentials — WiFi & API key stored in ESP32 NVS, never hardcoded

Hardware

  • ESP32-S3-DevKitC-1 — dual-core 240 MHz, built-in WiFi
  • MAX7219 FC-16 dot-matrix display — two chained 4×(8×8) modules
  • USB 5 V power supply
  • Optional: passive buzzer, DS3231 RTC, rotary encoder, RGB LED

Get Started

# 1. Clone
git clone https://github.com/kamioon/departure-countdown.git
cd departure-countdown

# 2. Flash
pio run -e esp32-s3-devkitc-1 --target upload

# 3. Enter WiFi + NS API key via serial
pio device monitor

# 4. Open the web UI
http://<device-ip>/

NS API key: apiportal.ns.nl → subscribe to Reisinformatie API.
